What is branching?

Git allows us to create different branches in a project. This way, we can work on a task independently without worrying about other changes made by the team. It also allows us to create code reviews by creating a pull request.

What is merging?

Merging is to push the code from one source branch to a target one. This is the step where we can identify possible conflicts.

What is a pull request?

It is a functionality that version control platforms like gitHub offer. This enables the team to easily see the new code changes when merging one branch to another, making it easier to review the code and identify possible problems.
